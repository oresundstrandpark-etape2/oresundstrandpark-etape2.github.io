Nyhed: www.selvaflæsning.com

På selvaflæsning.com kan jeres ejer/andels-forening selv lave årlige indrapporteringer af vand- og varmeforbrug, så jeres beboere ikke skal vente hjemme på en aflæser - og samtidig spare mange penge.

I kan dernæst også generere fordelingsregnskaber til jeres administrator - det er let, og I sparer endnu flere penge herved.

Selvaflæsning.com har flere kontroller indbygget end, hvad tilbydes af de fleste energiselskaber, hvilket gør kvaliteten højere end, hvad I hidtil har set. Resultatet er mere korrekte fordelingsregnskaber.

Husk at alle automatiske målere er krypteret, så hvis I overvejer at anskaffe dem fra et energiselskab, hvilket i sig selv er en større investering, så vil I være bundet til dem i mange år fremover.

Sig farvel til dyre aflæsninger og regnskaber fra Ista/Clorius, Brunata, Techem etc. og brug jeres foreningspenge på mere nyttige formål og projekter. Selvaflæsning.com bruger ikke penge på fine aflæsningsbiler, eksklusive domiciler og mange medarbejdere.

Funktioner

Selvaflæsning

Hver beboer modtager personlig login via email eller opslag i opgange/postkasser. Der kan så logges ind til hjælp og aflæsningsformular. Efterfølgende modtager beboeren email med aflæsningerne til bekræftelse.

Status

Bestyrelsen kan let se status med antal beboere, der har aflæst henholdsvis bekræftet. Der kan sendes rykkere per email til beboere, der ikke har bekræftet, eller endnu ikke foretaget selvaflæsning.

Regnskab

Fordelingsregnskaber kan genereres hurtigt for vand og varme. Systemet advarer, hvis der skulle mangle aflæsninger, aconto, forbrug etc.

Import / Export


Data kan let importeres fra tidligere energi-selskaber og administrator, samt efterfølgende let eksporteres til administrator eller andet energi-selskab.

Kontroller

Der er kontroller for blandt andet negative aflæsninger, forkert angivelse af komma, max/min/stdev på forbrug og forbrug per m2. Yderligere kontroller tilføjes løbende.

Hjælp

Der er vejledning til beboerne på dansk (og engelsk) med billeder og video. Beboerne og bestyrelsen kan kontakte support via mail 24x7. Telefonisk hjælp kan også arrangeres mod ekstra betaling.

Process

En hel ny og meget billigere måde at lave årlige aflæsninger og fordelingsregnskab.

Selvaflæsning

1. Import
Modtag beboerlist fra administrator og importer denne
Modtag acontolist fra administrator og importer denne
Modtag de tidligere aflæsninger fra energi-selskab og importer denne
2. Start aflæsningen
Systemet genererer login til hver beboer
Der kan printes opslag til opgangene, som bestyrelsen kan hænge op
Systemt kan sende email til beboerne med login eller der kan printes papirslip til omdeling
3. Aflæsning igang
Bestyrelsen kan checke status, og vente til 100% har selv aflæst
Systemet kan udsende rykker-email til beboere der ikke har selv aflæst eller bekræftet
Systemet kontrollerer aflæsningerne og angiver aflæsning- og indtastningsfejl

Fordelingsregnskab

4. Klar til regnskab
Modtag forbrugsregninger fra energi leverandør og angiv beløb til systemet
Systemet kontrollerer aflæsningerne igen og om alle oplysninger haves for alle beboere
5. Genererer regnskab
Systemet genererer regnskaber:
5.1. Fordelingsregnskab for vand — kold og varm
5.2. Fordelingsregnskab for varme
6. Eksport
Export fordelingsregnskab til administrator
Administrator opkræver som vanligt hver beboer med hensyn til for meget / for lidt betalt
