Github - Kun første gang

  1. Opret login til github
  2. Installer git værktøjer fra git-scm.com/download/win
  3. Alternativt download GitHub Desktop, hvis du hellere vil anvende GUI fremfor kommando-linje...
  4. Du skal nu lave en kopi af hjemmesiderne på din PC, hvor du kan lave ændringer, som du afsluttende sender tilbage. Kør denne kommando fra den sti, hvor du vil have den lokale kopi:

    F.eks. : "git clone https://github.com/oresundstrandpark-etape2/oresundstrandpark-etape2.github.io.git"

Dit brugerid på github skal sættes op til at være 'collaborator' før du kan tilføje dokumenter. Kontakt web administrator for at blive tilføjet.


Følgende trin for at uploade et nyt referat:

For pdf upload

    • placer pdf-filen i dokumenter-folderen som eksempelvis referat_YYYYMMDD.pdf

For word til html

  • Brug https://word2cleanhtml.com/ til at konvertere Word til HTML
  • Åben web-sitet, og dernæst Word-dokumentet. Markere alt i Word og copy-paste til web-sitet
  • Tryk 'Convert to clean html', og kopier indholdet tilbage til en ny html-fil, eksmpelvis kaldet dokumenter\referat_YYYYMMDD.html
  • Kopier html-header fra referat_20161101.html til nyt dokument
  • Sikre, at Encoding er UTF8, så æøå vises korrekt (brug eksempelvis NotePad++ værktøjet)

Opdater index.html

    • Tilføj en sektion til index.html med link til den nye referat_YYYYMMDD.html fil

Opdater referat.html

    • Tilføj en linje til referat.html med link til den nye referat_YYYYMMDD.html fil

Upload to github

    • Kør a.bat {kommentar} eller .\a.sh (mac/linux), eksempelvis "a.bat referat1nov"; eller brug git kommandoer (git add, git commit, git push som omtalt i andre kilder)

Post-check

    • Check hvordan det ser ud på f.eks. : https://oresundstrandpark-etape2.github.io - det tager et par minuter efter upload før siderne er opdaterede.


Ny hurtigere metode

Kræver installation af Python (https://www.python.org/downloads/) og Pandoc (https://pandoc.org/installing.html)

1. Placer Word og PDF dokumenter i Publish folder

2. Kør kommando : python publish.py EF-2-moede-07012019 20190107

3. Kør a.bat {kommentar} eller .\a.sh (mac/linux), eksempelvis "a.bat referat1nov"; eller brug git kommandoer (git add, git commit, git push som omtalt i andre kilder)

4. Check hvordan det ser ud på f.eks. : https://oresundstrandpark-etape2.github.io
