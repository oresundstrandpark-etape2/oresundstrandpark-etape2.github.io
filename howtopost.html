Andre kilder:

Github - KUN første gang

- opret login til github - installer git fra git-scm.com/download/win - du skal nu lave en kopi på din PC, hvor du kan lave ændringer, som du afsluttende sender tilbage. Kør denne kommando fra den sti, hvor du vil have den lokale kopi: "git clone https://github.com/oresundstrandpark-etape2/oresundstrandpark-etape2.github.io.git"

Dit brugerid på github skal sættes op til at være 'collaborator' før du kan tilføje dokumenter

Følgende trin for at uploade et nyt referat:

pdf upload

- placer pdf-filen i dokumenter-folderen som eksempelvis referat_YYYYMMDD.pdf

word til html

- brug https://word2cleanhtml.com/ til at konvertere Word til HTML. - Åben web-sitet, og dernæst Word-dokumentet. Markere alt i Word og copy-paste til web-sitet. - Tryk 'Convert to clean html', og kopier indholdet tilbage til en ny html-fil, eksmpelvis kaldet dokumenter\referat_YYYYMMDD.html. - Kopier html-header fra referat_20161101.html til nyt dokument - Sikre, at Encoding er UTF8, så ÆØÅ vises korrekt (brug eksempelvis NotePad2 værktøjet)

index.html

tilføj en sektion til index.html med link til den nye referat_YYYYMMDD.html fil.

referat.html

tilføj en linje til referat.html med link til den nye referat_YYYYMMDD.html fil.

upload to github

kør a.bat {kommentar}, eksempelvis "a.bat referat1nov"; eller brug git kommandoer (git add, git commit, git push som omtalt i andre kilder)

post-check

check hvordan det ser ud på : https://oresundstrandpark-etape2.github.io
